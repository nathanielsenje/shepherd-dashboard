⚡ CLAUDE CODE COMMANDS

After extraction, use these commands:

Extract Guidelines
──────────────────
python ui-guidelines-system/cli.py generate-guidelines \
  --input template.jpg \
  --output design-system.md

Generate Component
──────────────────
python ui-guidelines-system/cli.py build-component \
  --type button \
  --framework react \
  --css tailwind

Generate All Components
───────────────────────
python ui-guidelines-system/cli.py build-components \
  --guidelines design-system.md \
  --all \
  --output src/components/

Review Component
────────────────
python ui-guidelines-system/cli.py review-component \
  --file Button.jsx \
  --guidelines design-system.md \
  --fix

Validate Guidelines
───────────────────
python ui-guidelines-system/cli.py validate-guidelines \
  --file design-system.md

🆘 TROUBLESHOOTING

Problem: Python Not Found
────────────────────────
Solution: Install Python 3.7+
  macOS: brew install python3
  Windows: https://www.python.org/downloads/
  Linux: sudo apt-get install python3

Problem: Permission Denied
────────────────────────
Solution: Make scripts executable
  chmod +x install.sh
  chmod +x ui-guidelines-system/cli.py

Problem: Claude Code Not Found
─────────────────────────────
Solution: Install Claude Code
  npm install -g @anthropic-ai/claude-code

Problem: Can't Find cli.py
──────────────────────────
Solution: Check you're in the right directory
  cd your-project
  ls ui-guidelines-system/cli.py
